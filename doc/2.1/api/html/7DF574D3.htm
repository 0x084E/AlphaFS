[AlphaFS] Backs up (export) encrypted files. This is one of a group of Encrypted File System (EFS) functions that is intended to implement backup and restore functionality, while maintaining files in their encrypted state.

Remarks

The file being backed up is not decrypted; it is backed up in its encrypted state.

If the caller does not have access to the key for the file, the caller needs Backup to export encrypted files. See PrivilegeEnabler.

To backup an encrypted file call one of the ExportEncryptedFileRaw Overload overloads and specify the file to backup along with the destination stream of the backup data.

This function is intended for the backup of only encrypted files; see BackupFileStream for backup of unencrypted files.
