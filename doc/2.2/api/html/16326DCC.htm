Remarks
If there are no files or directories in the DirectoryInfo, this method returns an empty array. This method is not recursive. For subdirectories, the FileSystemInfo objects returned by this method can be cast to the derived class DirectoryInfo. Use the FileAttributes value returned by the Attributes property to determine whether the FileSystemInfo represents a file or a directory.
