ReparsePointTag Enumeration
Enumeration specifying the different reparse point tags.

 

Remarks

Reparse tags, with the exception of IO_REPARSE_TAG_SYMLINK, are processed on the server

and are not processed by a client after transmission over the wire.

Clients should treat associated reparse data as opaque data.
