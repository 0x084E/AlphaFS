    Parameters
    Type Name Description
    IntPtr winOwner

    true always pops-up an authentication dialog box.
    Boolean updateProfile

    true successful network resource connections will be saved.
    Boolean saveCredentials

    When the operating system prompts for a credential, the credential should be saved by the credential manager when true.
    Returns
    Type Description

    If localName is null or string.Empty, returns the last available drive letter, null otherwise.
