processSecurity
Type: SystemBoolean
Specifies whether the function will restore the access-control list (ACL) data for the file or directory. If this is you need to specify TakeOwnership and ChangePermissions access when opening the file or directory handle. If the handle does not have those access rights, the operating system denies access to the ACL data, and ACL data restoration will not occur.
